Result

data class Result : Reportable

Represents the result of an assertion or a group of assertions.

Properties

actual

 val actual: Any?

The actual value or values that violated the assertion. This property is optional as it does not make sense for all types of assertion. However, it can help improve diagnostic messages where it is appropriate.

cause

 val cause: Throwable?

The exception that caused the failure, if any.

description

 val description: String

The description of the assertion as passed to strikt.api.Assertion.assert.

expected

 val expected: Any?

The actual value, if relevant to the assertion.

message

 val message: String?

A description of the failure.

status

 val status: Status

The status of the result.
